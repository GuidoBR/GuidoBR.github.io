Curso de Python para aprender conceitos de Orientação a Objeto de forma divertida e prática. Recomendado para quem sabe pouco de Python e para quem quer reforçar os conceitos da OO trabalhando em um projeto prático.

Nesse curso constrói-se um clone simplificado do game Flappy Bird, com interface gráfica ou direto no terminal.

O processo é bem simples:

  1. Rode os testes (python3 executor_de_testes.py).

  2. Implemente um método que está quebrando um teste.

  3. Repita até todos os testes passarem.
