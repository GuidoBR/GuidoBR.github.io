Instaling Java with JavaFX Development Enviroment on Arch Linux

written by Guido Percú on August 18, 2016

Steps to install Java JDK + Java FX Scenebuilder + InteliJ IDE on Arch Linux (with yaourt to access AUR):

$ yaourt -S jdk
$ yaourt -S javafx-scenebuilder-bin
# pacman -S intellij-idea-community-edition
